# Powerbi-Consumer-Electronics-Analysis
Power BI Data Analysis Project
This Power BI project analyzes sales performance across products, categories, states, employees, and years. The dashboard highlights key insights such as yearly revenue trends, top-performing products, top states, employee performance, and factors influencing revenue growth.
The project includes:
Power BI Dashboard (.pbix)
Dataset (Excel/CSV)
Exported PDF report with visuals
The analysis helps identify high-revenue products, strong markets, and performance patterns to support better business decisions.
